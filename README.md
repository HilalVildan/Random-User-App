# Random User App

Create simple random user app with React

<hr />

### Installation Options

```
$ git clone https://github.com/HilalVildan/Random-User-App.git
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/HilalVildan/Random-User-App.git
```

```
$ yarn
```

```
$ yarn start
```
